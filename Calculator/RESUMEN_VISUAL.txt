┌─────────────────────────────────────────────────────────────────────┐
│                    🔬 FÍSICA CUÁNTICA IMPLEMENTADA                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ESTADOS CUÁNTICOS:                                                 │
│    |ψ⟩ = α|0⟩ + β|1⟩                                                │
│    |α|² + |β|² = 1 (normalización)                                  │
│    P(|0⟩) = |α|², P(|1⟩) = |β|²                                     │
│                                                                     │
│  PUERTAS UNIVERSALES:                                               │
│    ✓ Hadamard (H)  - Crea superposición                            │
│    ✓ Pauli-X       - NOT cuántico                                  │
│    ✓ Pauli-Y       - Rotación + fase                               │
│    ✓ Pauli-Z       - Cambio de fase                                │
│    ✓ CNOT          - Controlled-NOT (2 qubits)                     │
│    ✓ Toffoli       - CCNOT (3 qubits)                              │
│    ✓ SWAP          - Intercambio                                   │
│                                                                     │
│  ÓPTICA CUÁNTICA:                                                   │
│    E = hf         (Energía de Planck)                              │
│    λ = c/f        (Longitud de onda)                               │
│    h = 6.626×10⁻³⁴ J·s                                              │
│    c = 299,792,458 m/s                                             │
│    Espectro: 380-750 nm (luz visible)                              │
│                                                                     │
│  CIRCUITOS ARITMÉTICOS:                                             │
│    ✓ Sumador completo (Full Adder)                                 │
│    ✓ Sumador de propagación (Ripple Carry)                         │
│    ✓ Multiplicador (optimizado)                                    │
│    ✓ Operaciones: +, -, ×, ÷                                        │
│    ✓ Precisión: 8 bits (0-255)                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘


┌─────────────────────────────────────────────────────────────────────┐
│                      💡 EJEMPLO: 42 + 17 = 59                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  PASO 1: CODIFICACIÓN                                               │
│    A = 42 = 0b00101010                                              │
│    → Región A: |0⟩|1⟩|0⟩|1⟩|0⟩|1⟩|0⟩|0⟩                            │
│                                                                     │
│    B = 17 = 0b00010001                                              │
│    → Región B: |1⟩|0⟩|0⟩|0⟩|1⟩|0⟩|0⟩|0⟩                            │
│                                                                     │
│  PASO 2: CONSTRUCCIÓN DEL CIRCUITO                                  │
│    8 sumadores completos en cascada                                │
│    Cada uno: 3 CNOTs + 3 Toffolis = 6 puertas                      │
│    Total: 48 puertas cuánticas                                     │
│                                                                     │
│  PASO 3: PROCESAMIENTO (visualización en tiempo real)               │
│    Frame 1-8:   Puertas en bit 0, fotones propagando               │
│    Frame 9-16:  Puertas en bit 1, carry propagando                 │
│    Frame 17-24: Puertas en bit 2, estados cambiando                │
│    Frame 25-32: Puertas en bit 3, colores pulsando                 │
│    Frame 33-40: Puertas en bit 4, interferencia                    │
│    Frame 41-48: Puertas finales, resultado estabilizando           │
│                                                                     │
│  PASO 4: MEDICIÓN                                                   │
│    Output: |1⟩|1⟩|0⟩|1⟩|1⟩|1⟩|0⟩|0⟩                                │
│                                                                     │
│  PASO 5: DECODIFICACIÓN                                             │
│    Binario: 0b00111011                                              │
│    Decimal: 59 ✓ CORRECTO                                           │
│                                                                     │
│  ⏱️  Tiempo: ~32 ms (en simulación)                                  │
│                                                                     │
